# 基础设定

## 类型

杀戮尖塔 like 卡牌肉鸽，多人联机 PVE。

## 参考

- **核心玩法**：杀戮尖塔（Slay the Spire）
- **联机模式**：横跨方尖碑（Across the Obelisk）
  - 每人独立英雄 + 独立牌组
  - 共享敌人、同步回合、交错出牌
  - 战利品共享池、路线事件投票

## 核心设计决策

### 联机回合制的基础问题

四个人一起玩回合制卡牌，怎么不变成四倍等待时间。

横跨方尖碑的解法：**一张牌就切人**，不是 A 打完一整套才轮到 B。一个回合里四人的出牌交错进行，敌人只在全出完后行动。

### 联机的核心价值

Card combo 从"一个人凑够条件"变成了"两个人合起来凑够条件"。设计重心不是各自打怪，是跨玩家的状态引用——战士挂破甲→法师上火→游侠触发三倍。

### 职业不重叠

每人独立职业，技能方向不重叠。这是横跨方尖碑的核心约束——不是为了"多样性"，是为了支撑跨玩家 combo：破甲必须等战士打、燃烧必须等法师挂。
